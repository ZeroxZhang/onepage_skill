Onepager 是一个 Agent Skill，能够将用户提供的文字、Markdown 或 PDF 内容，通过渐进式交互流程，转化为精美的 OnePage 信息图（单页可视化海报）。

Applicable Scenarios
产品功能介绍 策略/方法论展示 信息图表制作 数据可视化概览 一页纸汇报
License: Apache License 2.0 Compatibility: Claude.ai / Claude Code / Trae

功能概述

Onepager 是一款面向 AI Agent 的信息可视化技能插件。它接收文字、Markdown 或 PDF 内容作为输入，通过 5 阶段渐进式交互流程（内容分析 → 配置选择 → 内容重构 → HTML 生成 → 截图交付），自动生成像素级精美的单页信息图。

整个过程无需用户编写任何代码，只需选择尺寸、风格和密度三个维度的配置，即可获得专业级的可视化输出。支持 Claude.ai 上传、项目集成和全局安装三种部署方式。

核心特性

多尺寸支持纵向长图 / 横向 16:9 / 正方形 1:1
四种设计风格科技极客 / 现代商务 / 清新自然 / 新拟态
三级信息密度低/中/高密度，自动改写内容匹配版式
智能图表匹配根据内容逻辑自动选择图表类型
中文字体优化根据风格匹配最佳中文字体方案
HTML → PNG自带 Playwright 截图脚本转高清图

安装方式

方式一：Claude.ai 上传
将 onepager/ 压缩为 ZIP → 设置 → 自定义 → Skills → 上传
方式二：项目集成
复制到项目目录下：
cp -r onepager/ .trae/skills/onepager/
方式三：全局安装
复制到个人配置目录：
cp -r onepager/ ~/.trae/skills/onepager/

使用方式

直接向 AI 描述需求即可自动触发：

"帮我把这段产品介绍做成一张信息图"
"用这份 Markdown 生成一个 OnePage"
"把这段文字做成适合手机阅读的长图"
斜杠命令：/onepager

依赖 & 兼容性

截图功能依赖（可选）

Python 3.8+ — 运行截图脚本
Playwright — pip install playwright
Chromium — python -m playwright install chromium

如果运行环境不支持截图工具，Skill 将直接交付 HTML 文件，Agent 会在需要时自动安装依赖。
