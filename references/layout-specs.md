# 尺寸与版式规范

本文件定义了三种尺寸模式的完整版式规范。

---

## A1: 纵向长图 (Portrait / Scroll-Friendly)

### 画布参数
- 宽度：800px（固定）
- 高度：根据内容自适应（不设上限，通常 1200-3000px）
- 外边距（页面 padding）：48px 左右，40px 上下

### 版式结构（从上到下）
```
┌──────────────────────────────────┐
│          顶部区域 (Header)         │  ← 主标题 + 副标题 + 元信息
│  渐变/装饰背景条                    │
├──────────────────────────────────┤
│          核心摘要区                 │  ← 1-2 句话概述（可选）
├──────────────────────────────────┤
│          内容模块 1                 │  ← 小标题 + 正文/图表
├──────────────────────────────────┤
│          内容模块 2                 │
├──────────────────────────────────┤
│          内容模块 3                 │
├──────────────────────────────────┤
│          ...更多模块...             │
├──────────────────────────────────┤
│          底部区域 (Footer)          │  ← 来源/署名/日期
└──────────────────────────────────┘
```

### 排版特点
- 单栏为主，信息从上到下线性流动
- 高密度模式下可使用双栏并排卡片
- 模块间距：32-48px
- 图表宽度可占满内容区或 80% 居中

---

## A2: 横向宽图 (Landscape / Presentation)

### 画布参数
- 宽度：1920px
- 高度：1080px（固定）
- 外边距：64px 左右，48px 上下

### 版式结构
```
┌─────────────────────────────────────────────────────┐
│  左侧标题区 (30%)   │   右侧内容区 (70%)              │
│                     │   ┌────────┐ ┌────────┐        │
│  主标题             │   │ 模块 1  │ │ 模块 2  │        │
│  副标题             │   └────────┘ └────────┘        │
│  摘要               │   ┌────────┐ ┌────────┐        │
│                     │   │ 模块 3  │ │ 图表    │        │
│                     │   └────────┘ └────────┘        │
└─────────────────────────────────────────────────────┘
```

**备选布局**（根据内容选择）：
- **顶部标题 + 下方多栏**：标题占顶部 20%，内容区使用 2-3 栏
- **左右分栏**：左侧 30% 为导航/标题，右侧 70% 为内容

### 排版特点
- 必须使用多栏布局（2-3 栏），禁止单栏
- 内容需高度紧凑以适应固定高度
- 字号可比纵向长图略小 1-2px
- 模块间距：24-32px

---

## A3: 正方形 (Square / Social Media)

### 画布参数
- 宽度：1080px
- 高度：1080px（固定）
- 外边距：48px

### 版式结构
```
┌───────────────────────────────┐
│         主标题区               │  ← 上方 15-20%
├───────────────────────────────┤
│    ┌─────────┐ ┌─────────┐   │
│    │ 模块 1   │ │ 模块 2   │   │  ← 中间 60-70%
│    └─────────┘ └─────────┘   │    2×2 或 2×3 网格
│    ┌─────────┐ ┌─────────┐   │
│    │ 模块 3   │ │ 模块 4   │   │
│    └─────────┘ └─────────┘   │
├───────────────────────────────┤
│         底部信息               │  ← 下方 10-15%
└───────────────────────────────┘
```

### 排版特点
- 空间极其宝贵，优先精简内容
- 建议使用 2×2 或类似的网格布局
- 图表需要缩放至适合单个网格格子
- 字号需适当缩小，但确保可读性（正文不小于 14px）
